## File Format
- **Format**: MP3 (best browser support)
- **Bitrate**: 128-192 kbps (good quality, small size)
- **Volume**: Normalized (not too loud)

## Alternative: No Sound Version
If you don't want to add sounds, the website will work perfectly fine without them. The sound toggle will simply not play anything.
